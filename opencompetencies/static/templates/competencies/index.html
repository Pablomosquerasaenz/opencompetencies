Welcome to Open Competencies

(pre-alpha demo version)

This is a bare-bones demonstration of what I'd like Open Competencies to be. Please kick the tires a bit, and share some feedback if you'd like to see this project fully developed.

You can start by looking at some sample competencies from Sample High School. Then go back and create your own school; once you create your own school, you will be able to fork an existing school's system. This forking behavior is one of the core features of the Open Competencies project.

You can also check out some sample pathways. A pathway is a collection of related elements of the competency system. You can check out a pathway that represents minimum graduation requirements for high school students, and a pathway that showing what a student who wants to become a physicist should focus on. This is another main feature of Open Competencies: the ability to identify different graduation requirements for individual students, based on their post-high school plans.

If you are a developer, you can check out the code from github. You can share feedback there as well.

If you'd like to get in touch directly, you can find me on twitter @ehmatthes, or you can email me at ehmatthes at gmail.
