Welcome to Open Competencies

(pre-alpha demo version)

To kick the tires, which are very tiny and wobbly at this early stage, click on the link below to view existing schools.

Poke around the existing schools briefly, then go back and create your own school. If you create your own school, you will be able to fork an existing school's system. This is a demo of the first main feature of the Open Competencies project.

You can also check out a sample pathway. A pathway is a collection of related elements of the competency system. One use of a pathway is to represent all of the learning targets required for a certain career. Each pathway belongs to a particular school, so there is no direct link to all pathways.

If you are a developer, you can check out the code from github. You can share feedback there as well.
